# Haxball V6 Real Soccer Setup Bot

## Türkçe

Haxball için geliştirilmiş V6 Real Soccer haritasına entegre bir script.

### Oda nasıl başlatılır?

1. https://www.haxball.com/headless adresine gidin  
2. Tarayıcı konsolunu açın  
3. HTML değerini `top` olarak ayarlayın  
4. Script kodunu yapıştırın  
5. Enter tuşuna basın  

### Gereksinimler

- Haxball Headless Host
- Doğru harita yapılandırması

Not: Botun düzgün çalışması için harita isimlerinin kod içerisinde tanımlanan isimlerle birebir aynı olması gerekmektedir.

## English

A script integrated with the V6 Real Soccer map for Haxball.

### How to start the room?

1. Go to https://www.haxball.com/headless  
2. Open the browser console  
3. Set the HTML to `top`  
4. Paste the script code  
5. Press Enter  

### Requirements

- Haxball Headless Host
- Proper map configuration

Note: For the bot to work correctly, make sure that the map names exactly match the ones defined in the code.
